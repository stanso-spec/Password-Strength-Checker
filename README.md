# Password Strength Checker

## Description
A simple command-line Password Strength Checker developed in Python that evaluates the strength of a password based on predefined security criteria. The program checks for minimum length, uppercase letters, numbers, and special characters, then classifies the password as Strong or Weak.

## Features
• Password strength validation  
• Minimum length check  
• Uppercase letter detection  
• Number detection  
• Special character detection  
• Strong/Weak password classification  

## Technologies Used
• Python 3  
• Regular Expressions (Regex)  
• String Manipulation  
• Conditional Statements  

## How to Run

1. Run the program:
   python main-4.py
   ```

2. Enter a password when prompted.

3. View the password strength result.
